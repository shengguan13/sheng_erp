# 项目总述
* 基于华夏erp：http://www.huaxiaerp.com
* 公司：吉林宏泽汽车部件有限公司
* 开发者：管盛

# 技术框架
* 核心框架：SpringBoot 2.0.0
* 持久层框架：Mybatis 1.3.2
* 日志管理：SLF4J 1.7
* 前端框架：Vue 2.6.10
* UI框架: Ant-Design-Vue 1.5.2
* 模板框架: Jeecg-Boot 2.2.0
* 项目管理框架: Maven 3.2.3

# 开发环境
建议开发者使用以下环境，可以避免版本带来的问题
* IDE: IntelliJ IDEA 2019.2+和JetBrains WebStorm 2019.3+
* DB: Mysql 5.7.33
* JDK: JDK 1.8
* Node: Node 16.16.0
* Maven: Maven 3.2.3+
* Redis: 6.2.1
* Nginx: 1.12.2
